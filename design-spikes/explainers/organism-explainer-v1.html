Omnilith in plain language

Think of an organism as a living creative thing.

A song can be an organism. A conversation can be an organism. A map can be an organism. Each one has a lasting identity, changes over time through state, and can contain other organisms through composition.

1. Coherence is noticed

Someone feels that this thing is real enough to be tended.

2. Threshold is crossed

A steward introduces it to Omnilith and takes responsibility for its life.

3. Identity is maintained

It keeps evolving through proposals, integration, and ongoing care.

Identity

Identity is the thread that stays the same even when the details change. You can always point to this organism and know it is still itself.

State

State is each moment of the organism's life. States are never overwritten, so the history remains visible and trustworthy.

Composition

Composition is how organisms live inside one another. A parent creates a boundary, and children inside it create richer behavior together.

Example: A release organism

Imagine a release as one organism. Inside it you compose song organisms, image organisms, policy organisms, and thread organisms. The whole release gains its own identity, while each child remains alive and independently tendable.

Vitality reflects recent tending and change.

Omnilith is an ecology: infrastructure provides the physics, and organisms provide the life.
